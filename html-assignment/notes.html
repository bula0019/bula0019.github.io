5 HTML Tags I have learnt

Here are five essential HTML tags that form the foundation of web development:

<div>
The <div> tag is a container element used to group and style sections of content on a webpage. It helps in organizing the layout and applying CSS styles.
<body>
The <body> tag contains all the visible content of a webpage, such as headings, images, forms, navigation, paragraphs etc.
<header>
The <header> tag defines the introductory section of a webpage or a section within it. It typically contains headings, logos, and navigation links.
<footer>
The <footer> tag represents the footer section of a webpage or a section. It usually contains information like copyright details, contact information, and links to related resources.
<nav>
The <nav> tag is used to define a section of navigation links. It helps in creating menus and facilitating user navigation throughout the website.
